Casier à Téléphones

Description du Projet

Ce projet consiste à transformer un lecteur DVD d'un vieux ordinateur en casier pour ranger les téléphones. Chaque casier est contrôlé par une carte Arduino et utilise la technologie NFC pour l'ouverture et la fermeture. Le système comprend un capteur de présence pour détecter si un téléphone est à l'intérieur, ainsi qu'un écran LCD pour afficher l'état des casiers.

Composants Utilisés

  1. Arduino UNO: Carte de contrôle principale.
  2. Lecteur NFC: Pour lire les badges NFC.
  3. Écran LCD (16x2): Pour afficher les messages à l'utilisateur.
  4. Capteur de Présence Ultrasonique: Pour détecter la présence d'un téléphone dans le casier.
  5. Mécanisme de Verrouillage Électronique: Pour ouvrir et fermer le casier.
  6. Buzzer: Pour fournir un retour sonore (positif ou négatif).

Fonctionnement Général

1. Lecture du Badge NFC

Chaque casier est équipé d'un lecteur NFC. Lorsque l'utilisateur présente son badge NFC, le lecteur vérifie si le badge est valide.

2. Ouverture et Fermeture du Casier

  • Si le badge est valide, le casier s'ouvre et reste ouvert pendant 5 secondes, permettant à l'utilisateur de déposer ou récupérer son téléphone.
  • Après 5 secondes, le casier se referme automatiquement.

3. Affichage sur l'Écran LCD

L'écran LCD 16x2 affiche des messages pour guider l'utilisateur à chaque étape.

Messages affichés :

  • Badgez svp : Invite l'utilisateur à présenter son badge.
  • Ouverture : Indique que le casier est en train de s'ouvrir.
  • Déposez le téléphone : Demande à l'utilisateur de déposer le téléphone dans le casier.
  • Fermeture : Indique que le casier est en train de se fermer.

4. Détection de la Présence du Téléphone

Le capteur ultrasonique à l'intérieur du casier détecte si un téléphone est présent. Cela permet de connaître l'état du casier (occupé ou disponible). Si le casier est vide, cela n'est pas considéré comme une erreur ; cela signifie simplement que l'utilisateur a choisi de ne pas déposer son téléphone.

5. Indication par Buzzer

Le buzzer émet un son pour indiquer si le badge présenté est valide ou non :

  • Son positif : Badge valide.
  • Son négatif : Badge invalide.

Très utile pour les personnes malvoyantes.

Interface Utilisateur

Écran LCD

Le système utilise un écran LCD pour afficher l'état des casiers.

Significations : "C" signifie "Casier" suivie de 1 ou 2 pour le numéro de casier

"O" signifie qu'un téléphone est présent dans le casier (Occupé).

"D" signifie que le casier est vide (Disponible).

Exemple d'affichage :

Badgez svp
C1 : O | C2 : D

Conclusion

Votre casier à téléphones est maintenant équipé pour fournir une solution efficace et sécurisée pour le rangement des téléphones. Le système est facile à utiliser grâce à son interface claire et ses indications visuelles et sonores. Suivez cette notice pour comprendre le fonctionnement et l'utilisation de chaque composant du casier.
